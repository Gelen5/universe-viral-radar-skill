# 博主变现路径分析

沿着证据链分析，不要直接猜“卖课”或“接广告”。

## 商业证据链

1. 主页简介、置顶内容、合集和高频关键词。
2. 视频中的产品露出、案例、身份背书和结果展示。
3. 评论区关键词、自动回复、私信引导和资料诱因。
4. 主页店铺、橱窗、群聊、公众号、企业微信或外部入口。
5. 免费内容、免费资料、低客单、高客单和服务升级路径。
6. 哪些内容负责流量、建立信任、筛选客户和成交。

## 输出格式

- **明确观察到的产品和入口**。
- **推测的产品阶梯**，标注置信度和推测依据。
- **内容到成交的路径图**。
- **关键转化节点**：用户在哪一步可能流失。
- **可复制的商业动作**与**不可复制的资源**。
- **针对当前用户产品的适配建议**：只提出能在其时间、平台和产品条件下执行的动作。
