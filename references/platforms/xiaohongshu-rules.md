# 小红书规则

小红书的第一目标不是“拍得像短视频”，而是让用户愿意点开、搜索、收藏并相信这条内容对自己有用。

## 一、平台优先级

- 首要目标：点击、搜索、收藏。
- 开头重点：标题、封面、需求匹配。
- 内容重点：实用、可搜索、可保存。
- 传播重点：搜索与收藏驱动的持续流量。
- 转化重点：评论关键词、私信、主页承接。

## 二、核心判断标准

1. **标题是否先讲结果或人群**：核心关键词尽量前置，但不能堆词。
2. **封面是否传达单一利益点**：3:4 封面优先，文字尽量短，能一眼看懂。
3. **问题是否足够具体**：不是“如何赚钱”，而是“普通上班族下班后怎么做第一单”。
4. **内容是否可收藏**：有步骤、模板、避坑点、对照表、清单或案例。
5. **搜索意图是否明确**：用户会用什么词找这条内容，正文里是否自然覆盖。
6. **人设是否可信**：是否说明你为什么值得讲，哪怕只是局部经验。
7. **标签是否辅助分发**：标签服务于人群和场景，不是热点词乱贴。
8. **是否过度标题党**：标题承诺太大、正文落地太弱时要降结论。

## 三、常见问题

- 标题很猛，但正文没有交付具体方法；
- 关键词堆积，读起来像搜索垃圾页；
- 封面视觉太杂，一屏同时说三件事；
- 内容只有观点，没有步骤和样例；
- 商业引导太急，像销售话术而不是经验分享；
- 内容和平台常见审美不匹配，显得太硬、太吵或太像纯广告。

## 四、发布前专项检查

重点看：

- 标题长度是否可快速扫读；
- 核心关键词是否前置；
- 正文是否有保存价值；
- 封面是否适合 3:4 竖版展示；
- 标签组合是否围绕同一主题；
- 内容是否只是对同类爆款换壳。

## 五、结论原则

- 点击强但正文弱：判定为“小改后发布”或“建议重写”。
- 标题普通但内容强：优先重写标题和封面，不急着重做正文。
- 有价值但搜索性不足：补关键词、补小标题、补清单化表达。
- 商业意图过重：延后转化动作，让内容先成立。
