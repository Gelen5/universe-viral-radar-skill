# 证据规则

所有判断必须区分三种状态：

- **已观察事实**：来自视频画面、字幕、主页、公开数据、评论或产品入口。
- **有证据推断**：根据多个观察事实推导，必须写清依据。
- **未知**：现有材料无法判断，不能用行业常识补齐成事实。

最低要求：

1. 每个关键结论至少绑定一个可定位证据：时间戳、画面编号、原句、公开指标或主页元素。
2. 不把播放量高直接等同于内容质量高；先比较账号基线、发布时间、粉丝规模与互动结构。
3. 不承诺“照着做就能爆”。输出的是机制、适配条件和验证动作。
4. 不复制原作者完整标题、脚本、案例和独有经历。只迁移抽象结构与用户心理机制。
5. 涉及变现时，必须区分“明确看到的产品/入口”和“合理推测的商业模式”。
6. 输入不完整时，先输出证据缺口，再给低置信度初步判断。
