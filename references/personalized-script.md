# 个性化选题和脚本生成

生成前必须读取创作者档案。档案不足时，最多追问会显著改变结果的 3 个问题。

## 选题矩阵

分别生成：

- 流量选题：扩大精准曝光；
- 涨粉选题：建立稳定关注理由；
- 信任选题：展示过程、观点、案例和能力；
- 成交选题：自然连接用户问题与产品解决方案。

每个选题包含：目标人群、痛点、反差、承诺、证据来源、商业目的、适用平台。

## 脚本输出

重点选题默认给出 15 秒、30 秒和 60 秒版本：

- 前 3 秒口播；
- 屏幕文字；
- 分段口播；
- 画面或 B-roll；
- 节奏中断点；
- 证据展示；
- 自然 CTA；
- 封面短标题；
- 发布配文；
- 置顶评论；
- 评论关键词回复；
- 私信承接话术。

## 原创要求

- 至少加入一个用户独有资产：真实经历、观点、案例、产品过程或失败教训。
- 不得复用对标视频连续表达、独有比喻或完整论证。
- 写成用户本人能自然说出口的话，删除空话和夸张承诺。
