# 对标博主分析

## 先判断是否值得对标

按 0—10 分评价：

- 受众重合度；
- 账号阶段接近度；
- 近期活跃度；
- 爆款是否可重复；
- 内容是否可复制；
- 变现路径清晰度；
- 与用户能力、时间和表达方式的适配度。

分类：主对标账号、相邻参考账号、单条内容样本、暂不推荐。

## 账号内容地图

分析：

- 主页定位：帮助谁、解决什么、凭什么相信、下一步入口。
- 人设构成：身份、经历、价值观、可验证证据。
- 内容支柱：流量、信任、成交三类栏目。
- 高频选题：人群、场景、痛点、结果、争议角度。
- 爆款基线：中位数表现、异常爆款、爆款重复率。
- 表达系统：常用开头、结构、案例类型、时长、画面、CTA。
- 评论需求：问题、反对、期待、购买信号和未被满足的需求。

## 输出必须回答

1. 这个账号具体值得学什么。
2. 哪些资源、身份和经历普通人无法复制。
3. 它的爆款是偶然还是有稳定机制。
4. 用户应该只学哪 1—3 个动作，而不是什么都学。
